# Robot Game 개발 및 테스트 가이드

이 프로젝트에는 개발 및 테스트를 위해 특정 단계로 즉시 이동할 수 있는 유틸리티가 포함되어 있습니다.

## 테스트 도구 사용법

### 1. 브라우저 개발자 도구 (Console)
브라우저의 콘솔창(F12 > Console)에서 `goto()` 함수를 사용하여 원하는 단계로 즉시 이동할 수 있습니다.
- 예: `goto(5)` (5단계로 이동)

### 2. 키보드 단축키
게임을 실행 중인 상태에서 `Alt` 키와 숫자를 조합하여 단축키로 단계를 건너뛸 수 있습니다.
- **Alt + 1~9**: 1~9단계로 이동
- **Alt + 0**: 10단계로 이동
- **Alt + -**: 11단계(수료식)로 이동

## 주의 사항
- 이 기능은 개발 및 디버깅 목적으로만 제공됩니다.
- 실제 사용자에게 배포할 때는 해당 코드를 제거하거나 비활성화하는 것을 권장합니다.
